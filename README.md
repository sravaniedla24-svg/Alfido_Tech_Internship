 # Zomato Dataset Analysis

## Project Overview

This project analyzes the Zomato restaurant dataset to identify trends in restaurant ratings, cuisines, locations, pricing, and online ordering services. The analysis was performed using Python and Exploratory Data Analysis (EDA) techniques.

## Objective

The main objective of this project is to extract meaningful insights from restaurant data and provide recommendations that can help improve restaurant discovery and customer experience platforms.

## Dataset Source

Kaggle - Zomato Dataset

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Tasks Performed

1. Data Loading
2. Data Cleaning
3. Missing Value Analysis
4. Duplicate Removal
5. Exploratory Data Analysis (EDA)
6. Data Visualization
7. Rating Analysis
8. Cuisine Analysis
9. Location Analysis
10. Price vs Rating Analysis

## Visualizations Created

* Top Restaurant Locations
* Restaurant Rating Distribution
* Top Cuisines
* Online Order Availability
* Price vs Rating Analysis

## Key Findings

* BTM, HSR, and Koramangala are major restaurant hotspots.
* Most restaurants have ratings between 3.5 and 4.2.
* Online ordering services are widely available.
* Popular cuisines dominate customer preferences.
* Higher prices do not always result in higher ratings.

## Recommendations

1. Partner with highly rated restaurants.
2. Promote trending cuisines.
3. Provide location-based recommendations.
4. Highlight affordable high-rated restaurants.
5. Improve personalized restaurant suggestions.

## Conclusion

The project successfully analyzed restaurant data and generated insights regarding customer preferences, restaurant performance, and market trends. The findings can help food delivery and restaurant discovery platforms make better business decisions.
